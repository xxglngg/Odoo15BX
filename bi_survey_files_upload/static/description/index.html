Survey Single or Multiple File Upload Options Odoo Apps

Survey Multi File Upload odoo app adds single or multiple file upload options in survey. Users can add questions in survey as file upload option as answer, participant can upload file as survey answer id upload more then one file warning will show. If you enable the "Upload Multiple file" option then participants can add multiple files as attachments. Participants can view all uploaded attachments by clicking the "review your answer" button on survey. Users can also see all uploaded attachments on survey answers.

Features

File Upload Option on Survey

User can upload file for survey questions.

File Upload Warning

Show warning on uploading multiple file if not allowed to upload multiple file.

Single File Upload Options

Upload single file as attachment for survey questions.

Upload multiple file as attachment for survey questions.

Survey File Upload Question

On survey user can add questions as file upload type and allow user to add file as attachment answer.

User can select "File Upload" as "Question Type", User can also see question type view icon for file upload.

Upload File On Survey

On survey participants can see "Add File" options for "File Upload" question type, by clicking this button user can upload a file as attachment.

User can see selected file uploaded as question answer for survey.

Review Uploaded File

Once participants finish survey he/she can view uploaded file by clicking "review your answers" button and also download and view attachments by clikng file.

View Uploaded File on Answers

User can see uploaded files on answers.

Single File Attachment Warning

If user tries to upload multiple file for single file upload then warning will show.

Survey Multiple File Upload Question

On survey user can add questions as file upload type and allow survey participants to add multiple file as attachment answer.

User can select "File Upload" as "Question Type", User also need to enable " Upload Multipal file" options only then survey participants can also see question type view icon for file upload.

Upload Multiple File On Survey

On survey participants can see "Add File" options for "File Upload" question type, by clicking this button user can select and upload multiple files as attachment.

Review Multiple Uploaded File

Once participants finish survey he/she can view all uploaded multiple file by clicking "review your answers" button and also download and view attachments by clikng file.

View Uploaded File on Answers
